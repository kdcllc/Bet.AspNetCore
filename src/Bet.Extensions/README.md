## Summary

The goal of this repo is to provide with reusable `nuget` package that contains commonly used functionality i.e. `ValueStopwatch`.


## Usage

The following classes to support `Async` development:

- `ActionOrAsyncFunc`
- `AsyncExpiringLazy`
- `AsyncFunc`
- `AsyncLock`
- `ReloadToken`
- `ChannelReader`
